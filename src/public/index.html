Sudoku

Rules

  • Each tile has a value between 1 and 9 (inclusively)
  • Each tile must be unique with all other tiles in the same row, column, and 3x3 grid
  • You have completed the puzzle when you have filled in all of the squares successfully
  • There is only 1 fixed board currently
  • This is a front-end application that talks to a back-end API
  • Complete board can be seen here
